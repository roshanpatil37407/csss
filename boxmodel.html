CSS Box Model Properties

Basic Box

This box shows the default box model. Padding, border, and margin are applied around the content.

Content Area

The content area is padded with a background color and a border.

Padding

This box has extra space around its content due to padding.

Border

This box has a border around it. The border thickness affects the total size of the box.

Margin

This box has extra space outside its border due to margin.

Box-Sizing

This box uses box-sizing: border-box;, so padding and border are included in the total width and height.
